# 🛒 Add to Cart Project

This is a **Simple Add to Cart Web Application** built using HTML, CSS, and JavaScript.  
Users can **add products to the cart** from the home page, **increase/decrease quantity**, and click on a product to view its **details page**.

## ⚙️ Features
- 🏠 Product listing on the Home Page  
- ➕ Add products to the cart  
- 🔍 Open product details page  
- 🛒 View products in the cart page  
- ➕➖ Increase or decrease product quantity  
- ❌ Remove product from the cart  
- 💰 Calculate total price
